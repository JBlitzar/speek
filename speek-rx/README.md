# speek-rx
Bluetooth to 3.5mm adapter. 


## How to use

Plug the USB-C port in to your computer.


Flash:
```bash
cd firmware
pio run -t upload
```

Now you can run the usb-c port off of a power brick or whatever


Speaker should show up in bluetooth as "speek-rx." Enjoy!

Line out and headphone jack are labeled in silkscreen
